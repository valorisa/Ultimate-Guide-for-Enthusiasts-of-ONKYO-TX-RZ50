# Guide de Contribution

Merci de contribuer à ce projet ! Pour garantir une intégration fluide :

1. **Fork** le dépôt et créez une branche eat/nom-fonctionnalite.
2. **Documentez** tout changement technique avec des références [p.X].
3. **Respectez** le style Python PEP 8 et les docstrings Sphinx.
4. **Ouvrez une Pull Request** avec une description claire.

Pour les suggestions de documentation, utilisez le template doc_suggestion.md.
